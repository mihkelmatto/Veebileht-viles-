# Git/Github
Mis on Git?
Git on versioonihaldussüsteem, mis jälgib koodi muudatusi ja võimaldab meeskonnatööd. See salvestab projekti ajaloo ja laseb vajadusel tagasi minna varasemate versioonide juurde.

Kuidas kasutada?
- git init - loob uue repositooriumi
- git add . - lisab muudatused jälgimisele
- git commit -m "kirjeldus" - salvestab muudatused
- git push - saadab muudatused serverisse
- git pull - tõmbab teiste muudatused alla

GitHub lihtsustab:
- Visuaalne veebiliides koodi vaatamiseks
- Pull requestid meeskonnatööks
- Issues süsteem ülesannete haldamiseks
- Automaatne varundus pilves
- Näide: saad vaadata projekti ajalugu graafiliselt ja võrrelda erinevaid versioone ilma käsureata

# IDE

Me võime eraldi rääkida tekstiredaktoridest, mis on mõeldud kiirelt tekstide redigeerimiseks. Võrdluseks IDE on programm, mis pakub kõik asjad ühes kohas. Koodi kirjutamine, Testimine, Debugimine ja versioonihaldus on kõik programmi sisse ehitatud.

Nagu näiteks Sublime Text on väga hea programm teksti redigeerimiseks.
Samas Pycharm on väga hea IDE, millel on väga palju mugavusi juba sisseehitatud, mis võimaldab paremini programmerida.
