# GIT/Github

Git on versioonihalduse tarkvara, mis võimaldab lähtekoodil hoida struktureeritud viisil, et mitme arendajaga saaks seda lahendada.
Github on giti teenusepakkuja, kus saab luua enda giti reposid, kuhu koodi laadida.

# IDE

Esmalt rääkima tekstiredaktori ja IDE erinevusest.
Tekstiredaktor on väike, kerge ja lihtne tööriist, teksti muutmmiseks ning tal pole väga palju asju sisse ehitatud.
IDE kui selline on programm, kuhu on kõik sisse ehitatud. Sa saad programmi jooksutada, debuggida otse programmis. Lisaks kõik mugavusasjad on sinna sisse ehitatud nagu näiteks git.
Sa ei pea programmist välja minema et neid asju teha.
