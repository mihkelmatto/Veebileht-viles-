Git

Giti koduleht

Git on versioonihalduse tarkvara, mis võimaldab lähtekoodil hoida struktureeritud viisil, et mitme arendajaga saaks seda lahendada. Sellisel viisil lähtekoodi hoidmine on väga kasulik eriti suurtes organisatsioonides, kus väga paljud erinevad arendajad töötavad korraga lähtekoodi kallal.

Git-i installimiseks tasub järgida kas Atlassian-i juhendit või Githubi enda juhendit.

Linkidel on infot ka kuidas seadistada Git-i ning kuidas autentimist teha. Kui see on tehtud, siis saad juba giti kasutama hakata.

Giti kasutamiseks tasuks lugeda töövoo juhendit, kuidas tavaliselt git'i kasutatakse.

Github

Github-i koduleht

Github on giti teenusepakkuja, kus saab enda loodud giti reposid üles laadida ning teistega jagada. Kuigi Git'i saab lokaalselt kasutada, on see palju kasulikum, kui sa saad kergelt jagada oma koodi ning kui sa kaotad oma arvuti ära, siis kood on ikka alles.

Kiirjuhend

Siin on kõige tavalisemad käsud, mida giti kasutajal tuleks käsureal kasutada. See kindlasti pole lõplik ning neil kõigil on samuti lippe, mida saad määrata

Veebileht, kus saab Giti käskude kohta infot

Käsk Mida käsk teeb?
git help Näitab infot kõigi giti käskude kohta
git init Loob käesolevasse kausta uue giti repo
git clone <https://github.com/> Kloonib giti repo käesolevasse kausta
git pull Kontrollib ja tõmbab uued muudatused giti repost alla
git log Näitab giti commitidest logi, kus näeb autorit ja sõnumit
git add file.txt Lisa staging-usse file.txt faili, et seda hiljem saaks kasutada pushimiseks
git commit -m "Commiti sõnum!" Loob uue commiti koos -m antud jutumärkide vahelise sõnumiga
git push Lükkab käesoleva branchi koodi üles
